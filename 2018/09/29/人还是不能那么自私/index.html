人还是不能那么自私

发表于 2018-09-29 | 分类于 随笔 | | 阅读次数:

突然之间想写点什么

       又到了一年一度的国庆了，依然记得去年国庆回家时候的情景。去年国庆连着中秋一起放假，所以回家过的中秋节，今年国庆也一直纠结着要不要回去老家，就现在，思想还是很复杂的。

       小孩已经一岁半多了，母亲跟着我出来这边带小孩也有一年多一点了，慢慢也适应这边的生活了，也会带着小孩出去玩。虽然当年只读了五年级，但是普通话还是能说一些，能和别人交流，之前还会去跳广场舞，又教会她用智能手机，教会她使用电脑看电视，这点让我比较欣慰，至少没有一直嚷嚷着要回老家，不然这会让我很内疚。毕竟在我的思想里，父母没有义务帮子女带小孩的，这不是他们的责任，他们的责任就是把自己的子女抚养长大，但是在中国的这个社会下，很多人认为父母帮子女带小孩是应该的，是必须要执行的义务，而且带不好还会责怪自己的父母，这个现象还是普遍存在的。

       母亲跟着我出来这边其实自己就很自私了，因为我剥夺了她的自由，她在家里，还有自己的母亲（我的外婆也有90高龄了），还有自己的女儿和外甥已经外甥女。所以如果在家，可以去看望自己的母亲，自己的女儿会回家看望她。她也可以做自己的事情，虽然在农村可能就是种菜和种地，但是比带小孩轻松多了，而且也自由很多。如果我母亲过来这边，不接触任何智能设备，不能带小孩出去交流，她也会一直说着要回老家的，因为老家有邻居聊天，可以串门，不会像城市一样那么约束着，这个时候我也应该把她送回去，不能留在身边。她应该有自己的自由！

       其实之前几天，母亲也提到了自己一个人坐车回去，大家一起回去开销太大了，我们这边放假带着小孩，今年正好又是她外甥10岁，加上我舅舅说过我外婆行动有点不方便了不如以前了，自己不能做饭了，力气也慢慢变小了。其实自己想到这些，应该让我母亲回去看望看望自己的母亲，她们可以聊聊天。毕竟国庆也是比较有充足的时间，人生应该把空闲下来的时间留着陪陪家人。可能过几年时候，我的母亲就没有母亲了。

感慨

       年轻的时候就应该多挣钱存钱，以至于以后不会为钱而烦恼

-------------The End-------------
