github的使用

首先到github官网注册账号并且登录进去（有账号的可以省略注册过程）GitHub

这个时候只能创建公开项目，不能创建私有项目，要创建私有项目要付费（好像是7美元一个月吧）

这里值得注意的是：在 Repository name 中的内容必须为[你的github名字].github.io，最好生成的路径就是：https://github.com/[你的github名字]/[你的github名字].github.io.git 。接下来点击Create repository即可。这个时候我们完成第一步：创建github的仓库。

Hexo

Hexo 是高效的静态站点生成框架，她基于 Node.js。 通过 Hexo 你可以轻松地使用 Markdown 编写文章，除了 Markdown 本身的语法之外，还可以使用 Hexo 提供的 标签插件 来快速的插入特定形式的内容。

首先，你必须在电脑中安装Node.js，直接下载点击安装就可以了

关联个人域名

这个时候有的同学可能不想用github的域名，想自己整一个域名放进去。先去买一个域名，我是在阿里云上买的

GoDaddy

阿里云

没有账号的就先注册，然后登录进去，搜索自己想要的域名，没有占用就购买，占用了就换一个。然后我们点击首页的控制台

进入之后，点击左边菜单下的 （域名 -> 云解析DNS）,如下图

添加记录，记录值就是跟你刚开始建的仓库名一样（xxxx.github.io），主机记录中的@字符和www的字符表示输入网址的时候带不带www都能正常的打开该域名

然后在hexo/themes/landscape/source目录下新建CNAME文件，内容为自己域名.
（注意:这个CNAME文件必不可少，每次运行generate的时候会把这个路径下的东西copy到public下. ）

然后打开github下博客的仓库，找到setting按钮
